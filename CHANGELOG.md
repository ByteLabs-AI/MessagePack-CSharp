## [1.2.0] / 2026-04-15
- this is the initial commit of the project, which includes the basic structure and initial implementation of the MessagePack-CSharp library. The library provides efficient serialization and deserialization of data using the MessagePack format, which is a binary format that is more compact than JSON. This commit sets up the foundation for future development and improvements to the library.
